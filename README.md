# Security_Lab
Security Automation Lab - FOR RESEARCH AND AUTHORIZED TESTING ONLY
This repository contains Python scripts and Linux configurations developed for **educational purposes** and **authorized security testing**.

## ⚠️ Legal Disclaimer
The tools in this repository are intended for research and authorized penetration testing only. Accessing or scanning systems without explicit permission is illegal. The author (JoshYZ) is not responsible for any misuse of these scripts.

## Contents
* **Network Recon:** Python wrappers for Nmap to automate service identification.
* **Cryptography:** Tools for analyzing classical ciphers (Caesar, Vigenère).
* **Log Forensics:** Scripts for parsing and decoding obfuscated system logs.

## Learning Goals
I developed these tools to better understand:
1. The logic of Buffer Overflows and memory safety.
2. How to automate security workflows using the Python `subprocess` module.
3. The importance of network hygiene and service versioning.
